# CodeAlpha_Email_Extractor

##  Email Extractor

A Python automation script that extracts email addresses from a text file using Regular Expressions (Regex) and saves them to a separate file.

##  Features

* Extracts email addresses automatically
* Uses Regular Expressions (Regex) for pattern matching
* Reads data from a text file
* Saves extracted emails to a new file
* Simple and efficient automation solution

## Technologies Used

* Python
* Regular Expressions (re)
* File Handling

* `email_extractor.py`
* `sample.txt`
* `emails.txt`
